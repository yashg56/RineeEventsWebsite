Package Options

The Complete Package:

Starting at $2500, this package gives you the full experience from ideation to execution. We will be there every step of the way making sure that the event you vision is the event that we will bring to the table. The Partial Package:

Starting at $1500, this package is for those that want the creative freedom of their event and handling some things on their own, but want us to figure out the hard parts and the nitty gritty details. We will help you however you need while making sure we aren't stepping on anybodies toes.

The Last Minute:

Starting at $700, we have a package that eliminates all the day-of stress. You have all the details planned out, the food picked out, decor in mind, but the last thing you want is to be stressed out on the special day. We will handle every part of the execution so that all you need to do is have an amazing time. Always best to let the professional handle it.
